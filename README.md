### 3-2. 전체 데이터 활용 가격예측
#### 모델의 가격예측성능을 높이기 위해 다음과 같은 방법으로 수치형데이터 전처리를 시도
- 첫번째 시도1. 모든 범주형 데이터 더미변수화

- 세번째 시도3. 일부 컬럼 카테고리임베딩 + 더미변수화
